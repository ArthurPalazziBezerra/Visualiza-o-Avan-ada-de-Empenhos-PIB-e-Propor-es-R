# Visualiza-o-Avan-ada-de-Empenhos-PIB-e-Propor-es-R
Este projeto foi criado com o objetivo de explorar visualizações mais trabalhadas em R, utilizando paletas especiais e gráficos comparativos que destacam padrões importantes nos dados. A ideia não é apenas analisar os valores de Empenho, PIB e Proporção de Gastos, mas apresentar essas informações de forma clara, estética e intuitiva.
